# SamiChatbot 💖

A special Telegram chatbot designed for a princess. 👑✨

## Features
- **Princess Treatment**: Every response is sweet, soft, and loving.
- **Currency Conversion**: Helping her with her calculations in the most adorable way.
- **Supportive Commands**: Always there to help her.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Create a `.env` file based on the template and add your `TELEGRAM_BOT_TOKEN`.
4. Run the bot: `python main.py`.
